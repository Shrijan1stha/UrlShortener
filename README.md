A web application created using Django that allows users to create shortened URLs, track number of click, and generate QR codes for shortened URLs.

**Features**

1) URL Shortening: It helps to shorten long URLs into short and simple URLs.

2) Click Tracking: Keep track of the number of times each URLs is clicked.

3) QR Code Generation: Integrated with django-qr-code to generate instant QR codes for every shortened link.

4) Management Dashboard: A table-based interface where you can view, edit, or delete your shortened URLs.

**Tech Stack**

Framework: Django 4.x / 5.x

Database: SQLite (Default) Frontend: Bootstrap 5, HTML5, Library: django-qr-code
